# Oejia_wx
Odoo 的微信模块，提供了对微信公众号的接入与管理，实现了微信消息与Odoo聊天的无缝对接

# 特性
* 用户、组同步管理
* 用户消息对接live_chat 一对一实时聊天
* 全功能自定义菜单配置
* 各种返回消息类型的支持
* 灵活配置自动回复及匹配方式
